## General info

## Technologies
Project is created with:
* python == 3.7.5
* pandas == 1.0.3
* nltk
* textblob
* googletrans
	
## Setup
To run this project:

### Engine - Terminal mode

```
$ cd backend
$ make setup-env
$ make run-model
```

### Backend 

```
$ cd backend
$ make setup-env
$ make run-backend
```

### Frontend

```
$ cd frontend
$ make setup-env
$ make run
```
