## General info
Replacement of words for language learning.
	
## Technologies
Project is created with:
* python == 3.7.5
* pandas == 1.0.3
* nltk
* textblob
* googletrans
	
## Setup
To run this project:

### Backend

```
$ cd backend
$ make setup-env
$ make run
```

### Frontend

```
$ cd frontend
$ npm install
$ ng serve
```
